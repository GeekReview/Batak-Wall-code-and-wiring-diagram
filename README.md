# Batak-Wall-code-and-wiring-diagram
This is a source code for a DIY Batak Wall using an Arduino UNO R3 WITH A SHIFT RESISTER.

Instruction:

  Buttons to Arduino pins:
1. Button 1 to pin 2
2. Button 2 to pin 3
3. Button 3 to pin A0
4. Button 4 to pin A1
5. Button 5 to pin A2
6. Button 6 to pin A3
7. Button 7 to pin 11
8. Button 8 to pin 12
9. Button 9 to pin A4
10. Button 10 to pin A5
11. Button 11 to pin 0
12. Button 12 to pin 1

  LEDs to Arduino pins/shift resister:
1. LED 1 to pin Q0 (Shift resister)
2. LED 2 to pin Q1 (Shift resister)
3. LED 3 to pin Q2 (Shift resister)
4. LED 4 to pin Q3 (Shift resister)
5. LED 5 to pin Q4 (Shift resister)
6. LED 6 to pin Q5 (Shift resister)
7. LED 7 to pin Q6 (Shift resister)
8. LED 8 to pin Q7 (Shift resister)
9. LED 9 to pin 7
10. LED 10 to pin 8
11. LED 11 to pin 9
12. LED 12 to pin 10

  Arduino to Shift resister:
1. VCC to 5V Arduino
2. DS to pin 4 Arduino
3. OE to GND Arduino
4. ST_CP to pin 6 Arduino
5. SH_CP to pin 5 Arduino
6. MR to 5V Arduino
7. Q7S not use
8. GND to GND pin Arduino
